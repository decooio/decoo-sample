# File uplode demo(JS)

## Setting environment variables

```shell
# signature for api
$ export AUTH_SIGNATURE=c3Vic3RyYXRlLWNUTVdDO....MWY4Nw==
# api domain
$ export GATEWAY_DOMAIN=http://localhost:5050
```
## Install packages

```shell
$ npm install
```

## Upload file(folder)

```shell
$ node index.js /{LOCALPATH}
```
